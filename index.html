rss2tumblr by lejoe

A php script to push content from rss feeds to your tumblr blog.

Install

  1. Get the code. (See "Download" instructions below)
  2. rename conf/config.yml-dist to conf/config.yml and edit this config file
  3. rename conf/feeds.yml-dist to conf/feeds.yml and add your feeds to the file
  4. Use a cronjob to call rss2tumblr.php as many times as you want. The script will publish only the new data.

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/lejoe/rss2tumblr

FAQ

Q:
Why not using the built in tumblr feed importer ?
A:
It’s limited to 5 feeds
It’s slow (updates every hours at best)
You can’t import every feed to any post type
You can’t customize which feed data goes to which tumblr
Q:
Can I use this tool with another blog than my default one (other than the primary one)?
A:
No. Unfortunately the tumblr API doesn’t allow that for now.
